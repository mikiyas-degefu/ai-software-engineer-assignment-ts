# AI Software Engineer Assignment

This project contains a small TypeScript HTTP client with tests.

## How To Run

### Run tests with Docker

Build the image:

```bash
docker build -t ai-software-engineer-assignment-ts .
```

Run the test suite in the container:

```bash
docker run --rm ai-software-engineer-assignment-ts
```

### Run tests locally

Install dependencies:

```bash
npm install
```

Run the test suite:

```bash
npm test
```
